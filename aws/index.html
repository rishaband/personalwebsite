role

amazon web services is amazon's cloud computing platform. i will be joining the AWS Backup team, which is a centralized service that lets you back up data across AWS services. so things like EBS volumes, RDS databases, DynamoDB tables, and S3 buckets all from one place. more specifically, i'm joining the job orchestration team, which owns the system that actually drives those backup jobs to completion. that means pulling jobs off queues, distributing them across a fleet of workers, handling throttling and retries from downstream services, and making sure everything runs reliably at a very large scale. the team sits within the broader AWS Data Protection org, which is focused on keeping customer data durable and recoverable.
